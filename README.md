# Agni
> Agni: Roman Hangman - iOS game for Latin and other language learning

[![Swift Version][swift-image]][swift-url]
[![Platform](https://img.shields.io/cocoapods/p/LFAlertController.svg?style=flat)](http://cocoapods.org/pods/LFAlertController)

[swift-image]:https://img.shields.io/badge/swift-5.0-orange.svg
[swift-url]: https://swift.org/

<img src="/icon-rounded.png" align="left" width="200" hspace="10" vspace="10">

Agni is like hangman, with a Roman-themed twist: instead of a hangman, 
it's a sheep being sacrificed to the gods, and its up to you to stop it. 
How? By guessing English words, Latin words, or whatever language you add.

<p align="left">
Currently, Agni is under construction as I rework it for iOS 13.
</p>
